# Sample Plot

![sample plot](https://github.com/tghira16/gstyle/blob/6c01bb56b3c0dec048a839ed29057b611880ff7e/example_plot.png)


To use it just call `pubplot()` in the hold block, e.g.
```
hold on 
pubplot()
plot(x, sin(x))
hold off
```
